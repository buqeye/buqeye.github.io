BUQEYE Guides and Cheatsheets

  • BUQEYE Guide to Projection-Based Emulators in Nuclear Physics: All examples discussed in the Guide and more are available as interactive, open-source Python Code so that practitioners can readily adapt projection-based emulators for their own work.
  • The BUQEYE Cheatsheet for Pointwise Truncation Errors (based on Appendix A in arXiv:1904.10581).

Python packages

  • gsum: A Bayesian model of series convergence using Gaussian sums. The gsum package provides convenient classes that allow one to analyze the convergence pattern of Effective Field Theory (EFT) observables. Specifically, this is a conjugacy-based implementation of the statistical model described in Melendez et al. (2019), arXiv:1904.10581 [Phys. Rev. C 100, 044001 (2019)].
