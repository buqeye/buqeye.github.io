Python packages

  • gsum: A Bayesian model of series convergence using Gaussian sums. The gsum package provides convenient classes that allow one to analyze the convergence pattern of Effective Field Theory (EFT) observables. Specifically, this is a conjugacy-based implementation of the statistical model described in Melendez et al. (2019), arXiv:1904.10581.
